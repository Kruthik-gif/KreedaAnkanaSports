# KreedaAnkanaSports
Kreeda-Ankana is an Android-based sports management application designed for villages and local communities to organize sports activities efficiently through a real-time digital platform. The app helps teams book playground slots, schedule friendly matches, manage team participation, and connect with nearby teams using an interactive challenge board. It aims to promote sports culture, teamwork, youth engagement, and better utilization of local sports grounds.
⚽ The system provides a smart ground booking feature with live availability updates to avoid scheduling conflicts. Teams can create and accept match challenges, organize tournaments, and receive instant notifications regarding bookings, match timings, and score updates. Kreeda-Ankana also includes live score tracking, team management, and match history features to improve coordination and transparency during sports events.
